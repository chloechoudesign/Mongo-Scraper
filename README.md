# Scrape-News

## Overview
- Mongo Scraper is a web application that utilizes Mongoose and Cheerio to scrape articles from InVision Blog, and allows users to save articles and to make notes.

## Live Site
- https://stark-basin-30187.herokuapp.com/

## Technologies Used
- Mongoose/MongoDB
- cheerio
